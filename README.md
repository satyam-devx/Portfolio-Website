🎭 Interactive 3D Layer

- Real-time WebGL rendering using Three.js
- Dynamic depth + immersive background

🧪 Particle Name Engine

- Custom-built Canvas system
- Text → Interactive particles
- Mouse-reactive physics

⚡ Motion System
